# 全球信息日报（Daily Global Info）

这是一个每天自动抓取全球信息的网站示例，基于 Python 标准库 + SQLite（无第三方依赖）。

## 功能

- 每天自动从多个全球 RSS 源抓取新闻。
- 按地区聚合展示（全球、欧洲、亚洲、非洲、美洲）。
- 提供手动“立即刷新”按钮。
- 提供 JSON API：`/api/articles`。
- 自动清理 14 天前数据。

## 如何一览你创建的网站

### 1) 本机运行

```bash
python3 app.py
```

启动后在浏览器打开：

- 首页：`http://127.0.0.1:8000`
- 数据接口：`http://127.0.0.1:8000/api/articles`

### 2) 快速检查服务是否可用

```bash
curl -I http://127.0.0.1:8000
curl -s http://127.0.0.1:8000/api/articles | head
```

如果看到 `HTTP/1.0 200 OK`，说明网站已正常启动。

### 3) 局域网访问（同一网络设备）

服务默认监听 `0.0.0.0:8000`，因此同一局域网设备可通过：

```text
http://你的机器IP:8000
```

进行访问（例如 `http://192.168.1.10:8000`）。

## 结构

- `app.py`: 抓取逻辑、数据库逻辑、Web 路由、后台调度
- `static/style.css`: 样式
- `test_app.py`: RSS 解析测试

## 说明

默认 RSS 数据源来自 NYTimes 与 BBC，若网络环境受限可替换 `FEEDS` 中 URL。
